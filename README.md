# 🚀 EXPENSE TRACKER APP (DATA SCIENCE PROJECT)

### 📌 **PROJECT OVERVIEW**

* **Goal:** Building a data-driven application to track, categorize, and visualize personal or business expenses.
* **Problem:** Managing finances manually is difficult, and people often don't know where their money is going.
* **Solution:** A system that not only records expenses but uses **Data Analytics** to show spending patterns and budget insights.

---

### 💼 **INDUSTRY RELEVANCE (FINTECH)**

* **Financial Awareness:** Similar logic is used by apps like **CRED, Mint, and Splitwise** to help users manage money.
* **Spending Behavior:** Uses analytics to identify "leakage" (where unnecessary money is being spent).
* **Budget Forecasting:** Helps users plan their future savings based on historical spending trends.

---

### 🛠️ **CORE FEATURES**

* **Expense Categorization:** Automatically groups spending into categories like Food, Travel, Bills, etc.
* **Data Visualization:** Uses **Pie Charts** and **Bar Graphs** to show monthly and category-wise distributions.
* **Trend Analysis:** Tracks how spending increases or decreases over time.
* **Budget Alerts:** Logic to identify if a user is overspending in a specific category.

---

### 📂 **DIRECTORY STRUCTURE**

* `data/` — Stores expense logs in CSV or JSON format.
* `src/` — Python scripts for data processing and aggregation.
* `app/` — **Streamlit UI** files for the interactive expense dashboard.
* `plots/` — Saved visualization images (Charts & Graphs).
* `requirements.txt` — List of libraries like Pandas, Matplotlib, and Streamlit.

---

### 🚀 **HOW TO RUN THIS PROJECT**

**1. Clone the repository**

```bash
git clone https://github.com/dakshh-23/expense-tracker-app.git
cd expense-tracker-app

```

**2. Install dependencies**

```bash
pip install -r requirements.txt

```

**3. Launch the App**

```bash
streamlit run app/main.py

```
